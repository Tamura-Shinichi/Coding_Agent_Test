# Coding_Agent_Test
GitHub Copilotを用いたCoding Agentのテスト

## 概要
このリポジトリは、Pythonの多重ループとフロー制御のテストを目的としています。

## ファイル説明

### nested_loops.py
2重ループの基本例を含むPythonスクリプトです。

```python
list1 = [1, 2, 3]
list2 = [4, 5, 6]
for i in list1:
    for j in list2:
        print(i, j)
```

## 実行方法
```bash
python3 nested_loops.py
```
